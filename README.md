# Frontend - ReactJS + Vite

Đây là frontend của ứng dụng, được phát triển bằng ReactJS sử dụng Vite.

---

## 1. Yêu cầu hệ thống

- Node.js 20+  
- npm hoặc yarn 

---

## 2. Clone project

```bash
git clone https://github.com/kushinar09/game-center-ui
```

## 3. Cài đặt dependencies
cho `npm`
```
npm install
```
hoặc cho `yarn`
```
yarn
```

## 4. Chạy ứng dụng
cho `npm`
```
npm run dev
```
hoặc cho `yarn`
```
yarn dev
```
Ứng dụng sẽ chạy tại:

http://localhost:5174
